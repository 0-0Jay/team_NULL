## 개요 및 개발동기

###  프로젝트 개요
> **발달장애인 지원 행정 업무**를 대상으로 지원 신청, 상담, 계획, 결과 등 전 과정을 통합 관리하고
> **문서 중심의 행정 업무를 디지털화**하여 사례 이력과 진행 상태를 체계적으로 관리할 수 있는 시스템을 개발하였습니다.

- **Node.js** 기반 서버, **Vue.js** 프론트엔드, **MariaDB**를 이용한 실시간 데이터 처리 및 저장

###  개발 동기
- 발달장애인 지원 업무가 종이 문서와 수기 기록 중심으로 운영되는 경우
  업무 이력 관리 및 진행 상황 파악이 어려운 문제를 해결하고자 함
- 행정 업무의 체계적 관리와 효율성 향상을 위한 통합 지원 시스템 도입 필요

### 기대 효과
- 데이터 기반 의사결정 : 지원 현황과 사례 데이터를 기반으로 신속한 판단 가능
- 사례 이력 관리 강화 : 모든 지원 이력을 기록하여 **문제 발생 시 원인 추적 용이**
- 업무 효율성 향상 : 행정 부담 감소 및 지원 품질 개선

## 주요 기능

### 회원 관리
- 회원 가입 
- 로그인

### 자원신청관리
- 지원신청서 조사지 등록/수정
- 지원신청서 작성/수정/조회
- 지원계획서 작성/대기/승인/반려
- 지원결과서 작성/대기/승인/반려
- 상담내역 작성/조회
- 지원신청내역 조회

### 마이페이지 관리
- 내 정보 수정
- 지원자 등록/수정
- 기관 정보 수정 

### 시스템 관리
- 기관, 기관담당자, 기관관리자 조회/등록/수정/사용승인/비활성화

### 문의/FAQ 관리
-  FAQ 조회/등록/수정/삭제
- 문의 조회/등록/수정
- 문의 답변 작성/수정
